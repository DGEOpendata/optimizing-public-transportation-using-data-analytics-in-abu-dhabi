# Optimizing Public Transportation Using Data Analytics in Abu Dhabi

## Overview
This project focuses on utilizing public transportation data in Abu Dhabi to optimize services, reduce congestion, and enhance the commuting experience. By analyzing a dataset containing detailed utilization metrics, urban planners, researchers, and policymakers can gain valuable insights into transportation patterns and make data-driven decisions.

### Key Features:
1. Analyze public transportation ridership data.
2. Identify peak hours for optimized service schedules.
3. Visualize transportation trends to facilitate better decision-making.

## Dataset
The dataset contains the following key fields:
- **Time**: Timestamp of data collection.
- **Ridership**: Number of passengers during the specified time.
- **Feedback_Rating**: Passenger feedback on service quality.
- **Demographics**: Passenger demographics such as age and gender.

### Data Format
- **File Format**: CSV
- **Sample File Name**: `Abu_Dhabi_Public_Transportation_Utilization_Data.csv`

## Getting Started
### Prerequisites
To run the analysis script, ensure you have the following installed:
- Python 3.7+
- Pandas library
- Matplotlib library

### Installation
1. Clone the repository to your local machine:
   bash
   git clone https://github.com/your-repo-name/abu-dhabi-transportation-analytics.git
   
2. Navigate to the project directory:
   bash
   cd abu-dhabi-transportation-analytics
   
3. Install the required Python packages:
   bash
   pip install pandas matplotlib
   

## Running the Analysis
1. Place the dataset file (`Abu_Dhabi_Public_Transportation_Utilization_Data.csv`) in the project directory.
2. Run the script:
   bash
   python analyze_transportation.py
   
3. The script will generate a plot displaying peak hours of public transportation usage.

## Results
The analysis will produce a line chart showing the total ridership during each hour of the day. This visualization helps identify peak travel times, enabling better resource allocation and service optimization.
